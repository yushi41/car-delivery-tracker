## 🎯 プロジェクト概要

### 目的
中古車の納車待ち状況をシンプルかつ美しく管理する

### コンセプト
「世界をとるアプリ」として、Appleのような洗練された美しさを目指す

### 対象ユーザー
- 初期: 1人（開発者自身）
- 将来: 3人程度の社内運用

### 利用デバイス
- **メイン**: iPhone
- **サブ**: Windows PC
- **形式**: ブラウザベースのウェブアプリ

---

## ✨ 主な機能

### コア機能（MVP）
- ✅ 複数車両の管理
- ✅ 納車予定日のカウントダウン表示
- ✅ カレンダー形式での表示
- ✅ 担当者の割り当て
- ✅ 車両情報管理（車種、価格、購入日など）

### あると嬉しい機能
- 📆 Googleカレンダー連携
- 📝 メモ機能
- 🏢 販売店情報管理

### 将来の拡張候補
- 📸 車両写真の添付
- 📞 連絡履歴の記録
- 🔔 納車日通知
- 📊 統計情報

### フロントエンド
- **Next.js 14** (App Router) - Reactフレームワーク

- **Tailwind CSS** - スタイリング
- **shadcn/ui** - UIコンポーネント
- **Framer Motion** - アニメーション

### バックエンド・データベース
- **Vercel Postgres** / **Supabase** - データベース
- **Prisma** - ORM

### デプロイ
- **Vercel** - ホスティング・デプロイ

### 外部連携
- **Google Calendar API** - カレンダー連携

### 認証（将来）
- **NextAuth.js** / **Clerk** - ユーザー認証

## 📝 開発メモ

### なぜこの技術スタックを選んだか？
- **Vercel + Next.js**: 相性抜群、デプロイが超簡単
- **TypeScript**: バグを事前に防ぎ、開発効率UP
- **Tailwind CSS + shadcn/ui**: 美しいデザインを高速実装
- **Framer Motion**: Appleのような滑らかなアニメーション

### デザインで重視すること
- **ワクワク感**: 使っていて楽しい
- **すごい**: 友人に見せたくなる品質
- **洗練**: 無駄のない美しさ
- **直感的**: 説明不要で使える
